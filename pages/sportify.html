Sportify

A sports matching and social networking platform designed to connect users by interest and skill level, with a focus on simple, intuitive interactions and approachable user flows.

View Source

Overview

Sportify is a platform that pairs users with similar sports interests and skill levels. It was built to explore social matching flows, profile discovery, and an approachable interface for organizing recreational sports connections in a simple and engaging way.

Project Details

Role

Semester 2

Technologies Used

PHP MySQL User Flow Responsive Design

Features

  • Interest-based matching and user discovery
  • Profile and social interaction flows
  • Responsive layouts for mobile and desktop
  • Clean onboarding and navigation experience

Development Process

  1. Mapping the user journey and core interactions
  2. Creating wireframes and interface structure
  3. Implementing the matching and profile features
  4. Refining the responsive experience across devices
  5. Validating the flow and polishing the interaction design

Challenges & Solutions

Challenge

Creating a matching flow that felt intuitive and easy to scan.

Solution

Built clear profile summaries and simplified actions so users could quickly find compatible sports partners.

Challenge

Balancing social features with a lightweight interface.

Solution

Prioritized core match-making interactions and kept additional elements in secondary sections.
