Neru Meru E-commerce

Responsive e-commerce website for a pet products retailer, built with Bootstrap, JavaScript, and HTML/CSS to showcase accessible layouts and product flows.

HTML CSS JavaScript Bootstrap
View Source

Overview

Neru Meru is an e-commerce website for pet products created from scratch without templates, which made the project a meaningful challenge from the beginning. Building the site required careful planning to structure each page in a way that was easy to use, efficient to navigate, and visually clear for visitors. In the end, we developed a standard e-commerce layout that highlighted the brand’s products in a way that felt polished and functional.

Challenges & Solutions

Challenge

Ensuring the product layout remained usable on smaller screens.

Solution

Used Bootstrap grid utilities and responsive classes to adapt card layouts and navigation at every breakpoint.

Challenge

Keeping product detail interactions clear and consistent.

Solution

Designed focused detail sections with clear button hierarchy, consistent spacing, and accessible labels.
